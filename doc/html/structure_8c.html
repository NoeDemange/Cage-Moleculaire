Detailed Description

Functions for converting molecular structures to graphs and helper functions.

This file contains functions for converting a molecule and a shell structure to a graph. It also includes a helper function to create a new node. The functions here facilitate the conversion of molecular structures to graph representations for further processing and analysis.

Helper function to create a new node with specified properties.

This function creates a new Node with the given properties, including the 3D point, the cost from the starting point (g), and the heuristic cost to the goal (h). The function also computes the total cost (f) as the sum of g and h.

Parameters
pointThe 3D point for the node.
gThe cost from the starting point to the node.
hThe heuristic cost from the node to the goal.
Returns
The newly created Node with the specified properties.
