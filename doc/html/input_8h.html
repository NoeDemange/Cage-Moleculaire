Molecule_t* readInput_xyz ( char *  inputname)

Read a molecule from a file in XYZ format.

This function reads molecular data from a file in the XYZ format and initializes a molecule with it. The XYZ file should contain information about the atoms and their coordinates in 3D space.

Parameters
