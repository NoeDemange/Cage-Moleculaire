Compute the distance between two points considering obstacles.

This function calculates the distance between the start and end points, considering obstacles in the voxel grid using A* pathfinding algorithm. It uses voxel distance heuristics to compute the path distance.

Parameters
startPosThe starting position.
endPosThe ending position.
voxelGridThe 3D voxel grid.
vMapThe voxel map for storing distances and index heap information.
nodeHeapThe node heap for storing nodes during the algorithm.
Returns
The computed distance between the start and end points, considering obstacles.
