Detailed Description

NodeHeap and VMap Data Structure Implementation.

This file contains the implementation of the NodeHeap and VMap data structures and related operations. The NodeHeap represents a priority queue, and VMap is a 3D array of mapping structures used for storing distance g and node indices in the NodeHeap.
