Detailed Description

Structure Main Operations.

This file contains the declarations of functions related to main operations in the application. These operations involve initializing, creating, deleting, and managing the Main_t structure and its elements (Shell_t, Molecule_t). It also includes functions for managing the allocation and copying of Shell_t elements.
