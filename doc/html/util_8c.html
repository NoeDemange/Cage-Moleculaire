Detailed Description

Utility functions for mathematical calculations and geometric operations.

This file contains utility functions for various mathematical calculations and geometric operations. It includes functions for converting angles between degrees and radians, computing distances between points, normalizing vectors, computing angles between points in a triangle, and performing rotations around points and vectors.
